# 🌦️ Weather App

A responsive weather application built using React.js that provides real-time weather updates based on user location or search.

## 🚀 Live Demo
https://weatherapp-eta-navy.vercel.app/

## 🛠️ Tech Stack
- React.js
- HTML
- CSS
- JavaScript
- Weather API

## ✨ Features
- Search weather by city name
- Displays temperature, humidity, and weather conditions
- Toggle between °C and °F
- Responsive design for mobile and desktop
- Clean and user-friendly interface

## 📁 Project Structure
- `src/` – Main React components and logic  
- `public/` – Static files
