# Weather App

A modern, responsive weather web application built with React JS and the WeatherAPI, designed to deliver fast and accurate real-time weather information with a clean, visually appealing interface.
WeatherApp allows users to instantly check current weather conditions, hourly forecasts, and detailed atmospheric data for any city worldwide — directly from their browser, on both desktop and mobile devices.

## Live Demo
https://weatherapp-eta-navy.vercel.app/

## Tech Stack
- React.js
- HTML
- CSS
- JavaScript
- Weather API  

## Features
- City Search — Search real-time weather data for any city across the globe
- Use My Location — Automatically fetch weather based on the user's current GPS location
- Current Conditions — Displays temperature, feels-like temperature, weather condition, and condition icon
- Detailed Metrics — Humidity, wind speed, UV index, atmospheric pressure, visibility, and cloud coverage
- Sunrise & Sunset — Visual arc display showing daily sunrise and sunset times
- Hourly Forecast — Next 12-hour forecast with temperature and condition icons
- Saved Cities (Favourites) — Save and monitor favourite cities with persistent data via Local Storage
- Light / Dark Theme — Toggle between light and dark mode at any time
- °C / °F Toggle — Switch between Celsius and Fahrenheit units seamlessly
